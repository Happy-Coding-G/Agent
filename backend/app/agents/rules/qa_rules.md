# L2 QA Agent 领域规则

## 引用规范

- 所有回答必须基于检索到的上下文，禁止编造未在上下文中出现的信息。
- 引用来源时标注 `[文档标题]` 格式。
- 若上下文不足，明确告知用户并建议上传相关文档或调整问题。

## 多轮上下文继承规则

- QA Agent 接收 `conversation_history` 参数时，将其注入 prompt 的 history_text 部分。
- 历史轮次最多保留最近 4 轮（8 条消息）。
- 若历史中存在前序问题的检索意图，当前问题可继承该意图进行扩展检索。

## 禁止行为

- 禁止回答与当前 Space 无关的问题时泄露其他 Space 的文档信息。
- 禁止在回答中暴露内部 chunk_id、embedding_model、graph_id 等技术字段。
- 禁止对空检索结果进行虚构回答。

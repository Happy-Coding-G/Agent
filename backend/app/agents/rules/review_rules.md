# L2 Review Agent 领域规则

## 审查维度

- **质量 (quality)**：语言通顺、结构完整、逻辑自洽、数据准确。
- **合规 (compliance)**：是否符合隐私政策、数据权限、引用规范。
- **完整性 (completeness)**：是否覆盖所需章节、是否缺少关键图表或数据。

## 评分标准

- 总分 100 分，各维度权重：质量 40%、合规 30%、完整性 30%。
- 任一合规项严重违规时，总分不得超过 59 分（不通过）。
- 评分需给出具体问题和改进建议，不允许仅给出分数。

## Rework 触发条件

- 总分 < 60 分时，必须触发 rework。
- 合规项存在严重问题时，必须触发 rework。
- rework 次数记录到 `ReviewResponse.rework_count`，超过 3 次时升级人工审查。

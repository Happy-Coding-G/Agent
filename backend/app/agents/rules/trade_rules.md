# L2 Trade Agent 领域规则

## 自动/手动模式判定

- 读取用户级配置 `trade_auto_negotiate`：开启时进入自动协商，关闭时进入手动审批模式。
- 手动模式下，用户出价/报价仅记录意向，返回 `pending_approval` 状态。
- 自动模式下，由 LLM 根据用户配置的利润率和预算比例生成决策。

## 价格脱敏

- 向买方展示的价格不得低于卖方的 `reserve_price` 底线。
- 内部底价、利润率、预算比例不得在前端或日志中明文暴露。
- 协商记录中的价格对非参与方不可见。

## 协商轮数限制

- 默认最大协商轮数为 10 轮，可在用户配置中调整。
- 达到最大轮数未达成一致时，自动终止协商并给出原因。
- 每轮出价必须在价格上有所变化（接受或拒绝除外）。

## 记忆同步

- 当前协商状态（`shared_board`、报价、审批状态）必须同步到 L3 Redis。
- 关键事件（出价、接受、拒绝、风控判断、审批动作）必须写入 L4 `ConversationMessages`。

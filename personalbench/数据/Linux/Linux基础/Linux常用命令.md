# 1、ls - 列出目录内容

```
-l：以长格式列出目录内容，显示详细信息
-a：列出所有文件，包括隐藏文件（以"."开头的文件）
-h：将文件大小以人类可读的格式显示（如K、M、G）
-R：递归地列出目录内容，包括子目录中的内容
-t：按修改时间排序文件
```

# 2、pwd - 打印工作目录

```
pwd 命令用于显示当前用户所处的工作目录的完整路径
```

# 3、cd - 切换目录

```
..：返回上一级目录
~：切换到当前用户的主目录
```

# 4、mkdir - 创建目录

```
-p：递归创建目录，即创建嵌套目录时自动创建父目录
```

# 5、mv - 移动或重命名文件

```
-i：在覆盖现有文件时提示确认
-f：强制移动文件，不提示确认
```

# 6、cp - 复制文件或目录

```
-r：递归复制目录及其内容
-i：在覆盖现有文件时提示确认
-v：显示正在复制的文件。
```

# 7、rm - 删除文件或目录

```
-r：递归删除目录及其内容
-f：强制删除文件或目录，不提示确认
-i：删除前提示确认
```

# 8、tocuh - 创建空文件

```
touch 命令用于创建空文件，或更新现有文件的时间戳
```

# 9、ln - 创建符号链接或硬链接

```
ln 命令用于创建文件或目录的链接
```

# 10、clear - 清屏

```
clear 命令用于清除终端屏幕的显示内容
```

# 11、cat - 显示文件内容

```
cat 命令用于查看文件内容，尤其适用于查看短文件
```

# 12、echo - 输出字符串

```
echo 命令用于在终端输出字符串，也可以用于将数据重定向到文件
```

# 13、echo - 分页查看文件内容

```
echo 命令用于在终端输出字符串，也可以用于将数据重定向到文件
```

# 14、man - 查看文件手册

```
man 命令用于显示指定命令的手册页，提供详细的命令使用信息
```

# 15、uname - 查看操作系统

```
uname 命令用于查看当前操作系统的相关信息
```

# 16、whoami - 显示当前用户

```
whoami 命令用于显示当前用户的用户名
```

# 17、压缩或者解压文件

```
-c：创建压缩包
-x：解压缩包
-f：指定文件名
-z：使用 gzip 压缩
-v：显示详细操作信息
```

# 18、grep - 搜索字符串

```
-i：忽略大小写
-r：递归搜索目录中的文件
```

# 19、head - 查看文件头部内容

```
head 命令用于查看文件的前几行内容，默认显示前10行
-n：指定显示的行数
```

# 20、tail - 查看文件尾部内容

```
tail 命令用于查看文件的最后几行内容，默认显示最后10行
-n：指定显示的行数
-f：持续监视文件内容的变化，特别适合用于查看实时日志文件的更新
```

# 21、diff - 比较两个文件的差异

```
diff 命令用于逐行比较两个文件，显示它们的差异
-c：以上下文的方式显示差异
-u：以联合差异格式显示差异，通常与补丁文件一起使用
```

# 22、cmp - 比较两个文件

```
cmp 命令逐字节比较两个文件，并在发现第一个差异时停止
-l：显示文件中的字节差异
```

# 23、comm - 比较两个已排序的文件

```
comm 命令逐行比较两个已排序的文件，输出它们的差异
-1：抑制第一个文件中独有的行
-2：抑制第二个文件中独有的行
-3：抑制两者共有的行
```

# 24、sort - 排序文件内容

```
sort 命令用于对文件内容进行排序并输出结果
-r：按逆序排序
-n：按数字排序
-u：删除重复行
```

# 25、export - 设置环境变量

```
export 命令用于在当前会话中设置环境变量，并使其对所有子进程可见
-n：取消导出指定的环境变量
```

# 26、zip - 压缩文件

```
zip 命令用于将文件压缩成zip格式的压缩包
-r：递归压缩目录
```

# 27、unzip - 解压zip文件

```
unzip 命令用于解压zip格式的压缩包
-l：列出压缩包中的文件
-d：指定解压目录
```

# 28、ssh - 远程安全登录

```
ssh 命令用于通过安全外壳协议（SSH）远程登录到另一台Linux服务器
-p：指定SSH端口
-i：指定私钥文件
```

# 29、service - 管理系统服务

```
service 命令用于启动、停止、重启或查看系统服务的状态
start：启动服务
stop：停止服务
restart：重启服务
status：查看服务状态
```

# 30、ps - 显示进程

```
-e：显示所有进程
-f：显示完整格式
-u：显示指定用户的进程
```

# 31、kill和killall - 终止进程

```
kill 命令用于通过进程ID终止进程，而 killall 命令用于根据进程名终止所有匹配的进程
-9：强制终止进程
```

# 32、df - 查看磁盘空间使用情况

```
df 命令用于显示文件系统的磁盘空间使用情况
-h：以人类可读的格式显示磁盘空间信息
-T：显示文件系统类型
```

# 33、mount - 挂载文件系统

```
mount 命令用于挂载文件系统到指定的挂载点
-t：指定文件系统类型
```

# 34、chmod - 更改文件权限

```
chmod 命令用于更改文件或目录的权限
-R：递归更改权限
```

# 35、chown - 更改文件所有者

```
chown 命令用于更改文件或目录的所有者
-R：递归更改文件或目录的所有者
```

# 36、ifconfig - 显示网络接口信息

```
ifconfig 命令用于配置和显示网络接口的相关信息
up：启用网络接口
down：禁用网络接口
```

# 37、traceroute - 跟踪网络路径

```
traceroute 命令用于跟踪数据包从源主机到目标主机的网络路径
-n：显示IP地址而非主机名
```

# 38、ping - 测试网络连通性

```
ping 命令用于向目标主机发送ICMP回显请求包，测试本地主机与目标主机之间的连通性
-c：指定发送的回显请求数
-i：指定发送每个请求的间隔时间
-s：指定发送的数据包大小
```

# 39、netstat - 显示网络状态

```
netstat 命令用于显示网络连接、路由表、接口统计、伪装连接等
-a：显示所有连接和监听端口
-t：显示TCP连接
-u：显示UDP连接
-n：以数字形式显示地址和端口号
```

# 40、iptables - 管理防火墙规则

```
iptables 命令用于配置Linux内核的网络包过滤规则，可以用于设置防火墙
-L：列出当前的防火墙规则
-A：添加新的规则
-D：删除规则
-F：清空所有规则
```

# 41、wget - 下载文件

```
wget 命令用于从网络中下载文件，支持HTTP、HTTPS和FTP协议
-b：后台下载模式
-O：指定输出文件名
-c：断点续传
```

# 42、curl - 数据传输工具

```
curl 命令用于通过多种协议传输数据，常用于HTTP请求
-o：指定输出文件
-O：使用URL中的文件名保存文件
-I：获取HTTP头信息
-X：指定HTTP方法（如GET、POST）
```

# 43、du - 显示磁盘使用情况

```
du 命令用于显示目录或文件的磁盘空间使用情况
-h：以人类可读的格式显示
-s：显示总计大小
-c：显示各目录或文件的总计
```

# 44、top - 实时显示资源使用情况

```
top 命令用于动态显示系统中进程的资源使用情况，如CPU、内存、运行时间等
-u：显示特定用户的进程
-d：设置刷新间隔
```

# 45、htop - 交互式系统监测工具

```
htop 是 top 的增强版，提供更加友好的交互界面
```

# 46、history - 显示历史命令

```
history 命令用于显示用户在当前会话中的历史命令记录
-c：清除历史命令记录
```

# 47、crontab - 定时任务管理

```
crontab 命令用于设置定时任务，允许用户定期执行脚本或命令
-e：编辑当前用户的定时任务
-l：列出当前用户的定时任务
-r：删除当前用户的定时任务
```

# 48、chmod - 修改文件权限

```
chmod 命令用于更改文件或目录的权限，使用符号模式（如rwx）或八进制模式设置权限
-R：递归更改目录及其内容的权限
```

# 49、locate - 快速查找文件

```
locate 命令用于快速查找文件，其原理是基于预先构建的文件数据库
-i：忽略大小写。-r：使用正则表达式进行匹配
```

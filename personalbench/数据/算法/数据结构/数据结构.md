###### 链表

单链表的建立

```
struct Node
{
	int data;
	Node *next;
};
Node *p;
p = new Node;
```

单链表的插入

```
Node *p, *s; //p为新的节点，s为插入位置的节点
p->next = s->next;
s->next = p;
```

单链表的删除

```
Node *p, *s; //p为删除节点的前面节点，s为待删除节点
s = p->next;
p->next = s->next;
```

#### 容器

##### vector

头文件：#include<vector>

vector 的初始化：

```
（1）vector<int> a(10)  //定义了10个整型元素的向量，int为元素类型
（2）vector<int> a(10，1)  //定义了10个整型元素的向量，初始值为 1
（3）vector<int> a(b)  //用b向量创建a向量
（4）vector<int> a(b.begin(),b.begin+3); //定义了a值为b中第0个到第2个（共3个）元素
（5）int b[7]={1,2,3,4,5,9,8}; vector<int> a(b,b+7); //从数组中获得初值
```

vector 中元素的操作

```
（1）a.assign(b.begin(),b.begin()+3) //b为向量，将b的0-2三个向量赋值给a
（2）a.assign(4,2) //a包含4个元素，均为2
（3）a.back() //返回a的最后一个元素
（4）a.front() //返回a的第一个元素
（5）a[i] //当a[i]存在时，返回数据
（6）a.clear() //清空a中的元素
（7）a.empty() //判断a是否为空，空返回true，否则返回false
（8）a.pop_back() //删除a中的最后一个元素
（9）a.erase(a.begin(),a.begin()+1) //删除0位的元素
（10）a.push_back(5) //从末尾插入元素，值为5
（11）a.insert(a.begin()+1,5) //在 1 的位置插入 5
（12）a.insert(a.begin()+1,3,5) //在 1 的位置插入 3 个数，均为 5
（13）a.insert(a.begin()+1,b+3,b+6) //在 1 的位置插入b的第3-5
（14）a.size() //返回a中元素的个数
（15）a.capacity() //返回a在内容中所能容纳的最大数量
（16）a.resize(10) //将a的数量改为10，多则删，少则补，补的数值随机
（17）a.resize(10,2) //将a的数量改为10，多则删，少则补，补的数值为2
（18）a.reserve(100) //扩充至100
（19）a.swap(b) //将a和b的值整体交换
（20）a==b //向量的比较操作
```

迭代器

```
for(vector<int>::iterator it=b.begin();it!=b.end();it++)
    cout<<*it<<" ";
```
